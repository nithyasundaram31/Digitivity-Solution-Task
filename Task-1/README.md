## Reusable Table Component
- 1) I created a reusable table component in React where columns and data are passed as props.
- 2) I implemented global search using Object.values to filter data dynamically across all fields.
- 3) For sorting, I used state to track the selected column and sort direction, and applied custom  sorting logic using JavaScript’s sort method.
- 4) I handled numbers using subtraction and strings using localeCompare to ensure correct ordering.
- 5) I also made the table fully dynamic by rendering rows and cells using map, so it works with any dataset.

## TechStack
- React (Functional Components)
- Hooks (useState)
- JavaScript (ES6+)
- Tailwind CSS (for styling)
